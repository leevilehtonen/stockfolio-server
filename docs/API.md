**Register user**
----
  Registers new user to database

* **URL**

  /api/users/register

* **Method:**

  `POST`
  
*  **URL Params**

  None

* **Data Params**

  None

* **Success Response:**

  * **Code:** 200 <br />
    **Content:** `{ success : true, msg: msg}`
 
* **Error Response:**

  * **Code:** 200 <br />
    **Content:** `{ success : false, msg: msg, err: err }`
